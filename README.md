# Webpack Starter

Current webpack version: 4.34.*

## Installation
```
npm install
```

## Build Commands
```
npm run build
npm run watch
```

## Tests
Set your testing URLs in the `package.json` file's `testing.urls` object.

### Visual

```
npm run test:visual-init
npm run test:visual-reference
npm run test:visual
npm run test:visual-approve
```

### Accessibility

```
npm run test:a11y
```
